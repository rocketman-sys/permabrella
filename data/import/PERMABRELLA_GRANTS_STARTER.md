# PERMABRELLA_GRANTS_STARTER.md

## Purpose

This file is a starter content pack for the PermaBrella grants page:

Canonical import format for production data loads:
- `data/import/grants.csv`
- `data/import/grants_README.md`

https://www.permabrella.org/grants

The aim is to populate the page with a small, useful set of locally relevant grants for community members in the Northern Rivers who may be involved in:

- food security
- community building
- social outreach
- local resilience
- environmental repair
- workshops and education
- gardens, sharing systems, and grassroots initiatives

Do not turn this into a giant generic grants database.

Start with a curated list of strong local and regional opportunities, presented clearly.

---

## Recommended Page Approach

The page should feel like a practical community resource, not a bureaucratic wall of funding jargon.

For each grant, show:

- Grant name
- Best for
- Funding
- Who can apply
- Status
- Link

Optional tags:
- Food security
- Community building
- Resilience
- Environment
- Local council
- Philanthropic
- Circular economy

Status labels should be simple:
- Open now
- Closed — track next round
- Check current round

If exact round status or dates are uncertain, do not invent them.

## Recommended UI Structure

Use a clean card or list layout.

Each item should show:

- Title
- Short blurb
- Funding
- Status badge
- Best for line
- Optional tags
- External link button

Suggested badge styles:
- Open now = green
- Check current round = amber
- Closed — track next round = muted
- Ongoing resource = blue or neutral

Do not make users click into every card just to see whether a grant is relevant.

The page should be scannable.

## Content Tone

Keep descriptions:
- plain
- practical
- concise
- useful for community members, not grant professionals

Avoid:
- bureaucratic jargon
- inflated claims
- stale or uncertain details presented as fact

Where round details may change frequently, keep the on-page copy general and rely on the source link for current specifics.

---

## Important Accuracy Notes

Before publishing each listing, verify:
- the link still works
- the grant still exists
- the status label is reasonable
- the eligibility line is not overstated

Do not invent:
- exact closing dates
- exact funding caps
- eligibility categories

Unless confirmed from the source page.

Where uncertain, use:
- Check current round
- Recent rounds up to...
- Check current program details
